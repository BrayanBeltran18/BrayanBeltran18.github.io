PokeAPI

Desarrollé una Pokédex en Java que muestra las imágenes y características de cada Pokémon. Utilicé la PokeAPI, una API disponible en línea, para obtener datos actualizados y dinámicos, lo que me permitió integrar información en tiempo real sobre los Pokémon en la aplicación.

Videojuego Breakout

Junto a mi equipo de trabajo desarrollé un emocionante juego de Breakout completamente en Java, recreando la clásica mecánica de romper bloques con una pelota. Implementé efectos visuales, un control fluido de la paleta y una jugabilidad desafiante para mejorar la experiencia del jugador.

Roll-Based Access Control

Junto con un compañero, desarrollé un sistema de control de acceso basado en roles (RBAC) utilizando Java y SQL. En este sistema, se definieron tres roles principales: administrador, doctores y pacientes. Cada rol tiene permisos específicos, donde los administradores gestionan usuarios y configuraciones, los doctores acceden a la información médica y actualizan los historiales, y los pacientes pueden consultar sus propios datos médicos. La integración de Java permitió crear una interfaz intuitiva, mientras que SQL se encargó de gestionar la base de datos para asegurar el almacenamiento y acceso adecuado a la información.

Carrusel de Perritos

Desarrollé un carrusel interactivo de imágenes de perritos en el que los usuarios pueden ingresar cualquier raza de su elección. El programa, basado en JavaScript y una API de imágenes de perros, muestra fotos correspondientes a la raza seleccionada y las va rotando automáticamente cada 10 segundos. Esto permite que el usuario disfrute de una galería dinámica y continua de adorables perritos según su preferencia.

Sobre Mí
